Сайт для бронирования квартир в Алматы
Описание проекта:
Данный проект представляет собой веб-приложение для поиска, просмотра и бронирования квартир в городе Алматы. Пользователи могут просматривать доступные квартиры, добавлять их в избранное, оставлять отзывы и бронировать жильё. Также реализована система личного кабинета и административная панель для управления контентом и пользователями.

Основной функционал:
1. Шапка сайта (Header)
Навигация по основным разделам сайта
Авторизация и регистрация пользователей
Выход из аккаунта
Быстрый доступ к личному кабинету

3. Страница всех квартир
Отображение списка всех доступных квартир
Миниатюры изображений квартир
Поиск по различным критериям (цена, район, количество комнат и др.)
Сортировка (по цене, рейтингу, дате добавления и т.д.)
Добавление квартир в избранное
Краткая информация о каждой квартире

5. Страница квартиры
Просмотр всех фотографий квартиры
Полное описание и характеристики
Средняя оценка и отзывы пользователей
Возможность оставить отзыв и оценку
Добавление в избранное
Функция бронирования квартиры

7. Избранное
Список избранных квартир пользователя
Удаление из избранного
Сортировка списка

9. Забронированные квартиры
Отображение всех активных бронирований
Информация о дате и условиях брони
Возможность отмены бронирования заранее

11. Личный кабинет
Просмотр и редактирование личной информации
Отображение истории бронирований
Просмотр оставленных отзывов
Управление профилем пользователя

13. Админ-панель
Управление пользователями (поиск, просмотр базовой информации)
Блокировка и разблокировка пользователей
Ограничение пользователей в комментариях и отзывах
Управление квартирами:
Добавление новых квартир
Редактирование информации
Удаление
Архивация
Просмотр всех квартир и бронирований
Сортировка и фильтрация данных

Цель проекта:
Создать удобную и функциональную платформу для аренды квартир в Алматы, которая упрощает поиск жилья для пользователей и предоставляет владельцам/администраторам инструменты для управления объявлениями и бронированиями.
